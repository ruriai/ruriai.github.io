GitHub Pages域名托管到Cloudflare

2025-07-12 GitHub Pages ， Cloudflare ， Domain

GitHub为用户提供了USERNAME.github.io，用来做个人博客再合适不过。但.github.io着实有点……不够个性化，所以我打算将其托管到.kinoko.fun下

  • GitHub Pages提供：https://ruriai.github.io/

注册Cloudflare账号，并确保已经有domain托管到此账号上

在国内云服务商购买的域名也可以通过修改DNS解析的形式再托管到Cloudflare，可以参考该教程：腾讯云域名托管到 cloudflare - BigOrang - 博客园

为该域名添加 CNAME DNS解析

然后回到USERNAME.github.io仓库，在Setting -> Pages -> Custom domain中添加你自定义的域名，保存后稍等片刻，等提示DNS check successful!就可以了

  • 自定义域名：HammerCat - https://ruriai.kinoko.fun
